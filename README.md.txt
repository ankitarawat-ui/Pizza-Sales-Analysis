# Pizza Sales Analysis | SQL + Power BI

## Project Overview
This is an end-to-end data analytics project where pizza sales data was analyzed using SQL and visualized using Power BI. The goal of this project is to identify sales trends, customer ordering behavior, and product performance to support data-driven business decisions.

## Business Objectives
- Analyze overall revenue and order performance
- Identify best and worst-selling pizzas
- Understand sales trends by day and month
- Evaluate category-wise and size-wise performance
- Support pricing, promotions, and inventory planning

## Tools Used
- SQL
- Power BI
- Power Query
- DAX

## Dashboard Features
- KPI cards: Total Revenue, Total Orders, Average Order Value
- Sales trends by day and month
- Revenue breakdown by category and size
- Best and worst-selling pizzas
- Interactive slicers and filters

## Key Insights
- Weekends generate the highest number of orders
- Classic pizzas contribute maximum revenue
- Large-size pizzas are the most profitable
- Some pizzas underperform consistently

## Outcome
This project demonstrates a complete BI workflow from SQL-based data analysis to interactive Power BI dashboards for business decision-making.
